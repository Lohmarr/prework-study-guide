HTML

  • HTML is what structures the webpage.
  • An element is a type of content on a webpage. Commonly used elements are Text, Image, Header, and List.
  • An HTML attribute defines additional information about an element.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stylizes our webpage.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • pwd: displays the current working directory.
  • ls: displays the list of files and folders in the current directory.
  • cd: navigates between directories in the path.
  • cd .. : goes backwards from your selection.
  • git pull origin main: MAKE HABIT. Makes sure the local branch is up to date before pushing up changes. Pulls base branch to feature branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • One of the biggest problems with declaring variables with the 'var' keyword is that you can easily overwrite variable declarations. When using the 'let' keyword instead, a variable with the same name can only be declared once. When using the 'const' keyword there is an added bonus that it becomes read-only; a constant value that cannot be reassigned.
  • 'x++' is the short way of typing 'x + 1"
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
  • Arrays start at 0 instead of 1.
